## Design System
Always read DESIGN.md before making any visual or UI decisions.
All font choices, colors, spacing, and aesthetic direction are defined there.
Do not deviate without explicit user approval.
Key rules:
- Display font: Young Serif (for destination names, quorum counters, dashboard hero)
- Body font: Plus Jakarta Sans (for all UI chrome, buttons, body copy)
- Background: #F5F0E8 (warm ivory — not white)
- Accent: #A81B0A (deep crimson — not orange)
- No cold grays, no blue accents, no generic rounded card styles
In QA mode, flag any code that doesn't match DESIGN.md.
